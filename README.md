### 온라인 스토어

### User
- 이용자
  -  login x 혹은 admin 권한 x, 조회만 가능
- 관리자
  -  login 혹은 admin 권한
  - 주문내역(Order)
    - 조회, 생성, 수정, 삭제
    - 주문 내역 전체 조회
    - orders/order
    - 유저 상세 페이지에서 해당 유저 주문내역 조회
      - user/users/mypage
  - 상품(Item)
    - items/item
    - 조회, 생성, 수정, 삭제
    
  - 결제 관리
    - order의 status로 결제대기, 결제완료
  

### Item
- 상품 등록,수정,삭제
- 상품 주문
